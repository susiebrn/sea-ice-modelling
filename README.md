This project demonstrates a simple code to compute Sea Ice Volume with cdo using Sea Ice Concentration and Thickness.
The demo notebook shows some plots of Sea Ice around Antarctica and includes some animations (GIF). Additionally it provides a code to compute the trend per grid cell.
The data provided are only values for September. The source data can be downloaded  here: [Environment and Climate Change Canada - CMIP6 Scenarios](https://climate-scenarios.canada.ca/?page=cmip6-scenarios)
